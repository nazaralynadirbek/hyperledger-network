# hyperledger-network
Hyperledger Composer

## Usage

```
git clone https://github.com/naznadmn/hyperledger-network.git

cd hyperledger-network/server

npm install

npm run deploy
npm run ping
npm run server

node populate

cd ../client

open index.html
```

1. Participants: Customer, Vendor, Banker
2. Assets: Product
